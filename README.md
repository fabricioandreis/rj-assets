# RJ Assets

Repositorio publico para publicar os mockups estaticos do Recreio da Juventude.

## Conteudo

- `index.html` - landing page principal do GitHub Pages
- `mockup/` - mockups v1 e v2 do app mobile
- `header.jpg` e `footer.png` - assets de apoio existentes no repo
